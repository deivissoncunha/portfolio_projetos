Projetos em Ciência de Dados

Desenvolvimento de um Dashboard Gerencial em Python e Streamlit

Neste projeto, foram aplicados conceitos de programação em Python, extração, tratamento e análise de dados, aliados a pensamento analítico e lógica de negócio. Utilizando Streamlit para o desenvolvimento da interface web e GitHub para versionamento e deploy, foi construído um painel interativo de indicadores com as principais métricas operacionais de uma empresa marketplace de delivery de alimentos.

O produto final consiste em um dashboard hospedado em ambiente cloud, acessível via link web, permitindo a visualização dinâmica dos dados e o suporte à tomada de decisão baseada em evidências.

Ferramentas utilizadas:
